## 2-1 什么是 Typescript (05：54)
- TS
    + Vue
    + Ng
    + Antd
- State of JS 权威调查报告
    + 未来所有的项目都要使用 TS
    
- 编程语言的类型
    + 动态类型语言 (Dynamically Typed Language)

    + 兼容各种浏览器，各种系统，各种服务器，完全开源
